RESEARCH PAPER
SIGN LANGUAGE TO SPEECH CONVERTER
This paper presents a comprehensive study on recognizing and converting sign language to speech using machine learning models based on Convolutional Neural Networks (CNNs) with a vision-based approach and the OTSU method. Sign language is a vital means of communication for the deaf community, and bridging the communication gap between the deaf and hearing communities is crucial for inclusivity and accessibility. We propose an innovative approach that combines the power of CNNs, a vision-based approach, and the OTSU method for robust and accurate sign language recognition and translation into spoken language. ABSTRACT : datasets used : METHODOLOGY : This study focuses on creating and preprocessing a robust American Sign Language (ASL) dataset to train and evaluate machine learning models for sign language recognition and conversion. The dataset contains 27,455 grayscale images of hand signs (28x28 pixels), representing letters A-Z, excluding "J" and "Z," which require hand motion. The dataset ensures diversity in gestures, signers, and lighting conditions for generalizable performance.
Dataset Preprocessing involves three main steps:
  • Image Enhancement: Images are reshaped to a consistent 28x28 size for compatibility with the model and scaled from a 0-255 range to 0-1 to reduce processing time and improve edge details.
  • Noise Removal: Techniques such as Gaussian and median filters remove noise, while background subtraction isolates hand gestures from static backgrounds.
  • Label Binarization: The dataset labels are transformed into a numerical array format using a label binarizer, enabling faster image processing.
These preprocessing methods ensure the dataset is optimized for training robust neural networks in ASL recognition. This study presents the development of a Sign Language to Speech Conversion System using a Convolutional Neural Network (CNN) for gesture recognition. The CNN architecture includes convolutional layers for extracting spatial features, pooling layers for dimensionality reduction, fully connected layers for processing extracted features, and an output layer utilizing softmax activation for multi-class classification. The system is designed to handle complex visual patterns in sign language, making it highly effective for gesture recognition tasks.
The process begins with preparing a diverse, labeled dataset that captures variations in gestures, hand shapes, and expressions. Images undergo preprocessing steps such as enhancement, noise removal, and normalization to extract meaningful features. The CNN is trained iteratively using the labeled dataset, with hyperparameters like learning rate and batch size fine-tuned for optimal performance.  results : The evaluation of the Sign Language to Speech Conversion System highlights its high accuracy of 99.6% and a loss value of 0.92, demonstrating its effectiveness in gesture recognition. Precision metrics further validate the system's ability to correctly predict positive gestures. These results showcase the robustness of the CNN-based approach in recognizing sign language gestures with minimal errors.
For future improvements, potential enhancements include leveraging transfer learning with pretrained CNN models to improve accuracy, especially with limited data. Incorporating temporal modeling using RNNs, LSTMs, or TCNs could capture the motion dynamics of gestures for better recognition. Additionally, integrating natural language processing (NLP) could enable seamless translation of gestures into spoken or textual language, significantly enhancing communication between signers and non-signers. These advancements aim to make the system more versatile and effective.
