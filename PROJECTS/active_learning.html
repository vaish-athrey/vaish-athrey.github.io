RESEARCH PAPER
ACTIVE LEARNING
Active learning is a specific instance of supervised machine learning. The underlying principle of active learning is that an ML algorithm might potentially attain a higher degree of accuracy with a reduced number of training labels if it could choose which data it wanted to learn from. Using a modest amount of data with labels, the model is trained repeatedly for predicting labels of data that isn’t labeled. Next, the data points selected for categorization are those about which the algorithm is least certain. This iterative method is carried out till the required degree of precision is attained. By actively selecting the important data points, this method reduces the size of the dataset used for training while creating a high-performance classifier. Moreover, it also reduces the cost and time of dataset labeling while also being able to attain a high degree of precision. This paper examines the application of active learning using the Fashion MNIST, EuroSAT, and CIFAR10 datasets. The output of different active learning algorithms, such as Margin Sampling, Entropy Sampling, and Least Confidence, are compared. Furthermore, the model’s performance over random sampling is also demonstrated. It is seen that the accuracy obtained by using Active Learning is much higher as compared to random sampling. ABSTRACT : datasets used : METHODOLOGY : cifar 10 dataset fashion mnist dataset eurosat dataset This project explores active learning techniques using three benchmark datasets: CIFAR-10, EuroSAT, and Fashion MNIST.
  • CIFAR-10 features 60,000 low-resolution images across 10 categories, ideal for evaluating image classification models.
  • EuroSAT includes 27,000 satellite images spanning 10 land-use classes, advancing remote sensing analysis.
  • Fashion MNIST offers 70,000 grayscale images of apparel items, providing a modern alternative to MNIST for image recognition in the fashion industry.
These datasets enable the development and comparison of machine learning models in diverse domains. This project leverages active learning to enhance model training efficiency by strategically selecting the most informative samples for labeling. Beginning with a labeled training set, a deep learning model is pre-trained and used to process unlabeled data, generating feature maps that represent extracted features. Using query strategies like Least Confidence, Margin Sampling, and Entropy Sampling, the framework identifies data points with the highest uncertainty, ensuring that manual annotation focuses on samples that significantly improve model performance. These annotated samples are then added to the training set, and the model is retrained iteratively, refining its classification accuracy with each cycle.
The methodology drastically reduces the labeling workload by concentrating on critical data points, making active learning a cost-effective solution for handling large datasets. Techniques like Least Confidence, which measures prediction certainty, Margin Sampling, which evaluates confidence gaps between top predictions, and Entropy Sampling, which calculates uncertainty based on probability distributions, ensure optimal sample selection. This iterative approach not only improves model accuracy but also minimizes labeling resources, demonstrating its effectiveness in large-scale image classification tasks. ARCHITECTURE : This project integrates a Convolutional Neural Network (CNN) with active learning for efficient and accurate image classification. The CNN processes 32x32x3 images using three Conv2D layers with 3x3 kernels and ReLU activation to extract spatial features like edges and textures. MaxPooling2D layers reduce spatial dimensions, while dropout layers (25% and 40%) prevent overfitting. After feature maps are flattened, a Dense layer with 128 units (ReLU activation) learns the integrated features, followed by a Dense layer with 10 units and SoftMax activation to generate class probabilities.
By combining this CNN architecture with active learning, the system selects the most informative samples for labeling and iteratively updates the model, improving accuracy while minimizing the need for labeled data. This approach is cost-effective and ideal for scenarios where labeled data is limited, offering a robust solution for image classification tasks. results : The results demonstrate that active learning significantly improves accuracy compared to traditional training methods across the CIFAR-10, EuroSAT, and Fashion MNIST datasets. Accuracy from traditional training is 70.49%, 85.94%, and 86.11% for the respective datasets. Using active learning methods:
  • Least Confidence achieves 71.5%, 83.8%, and 87.2%.
  • Margin Sampling yields 71.4%, 86.09%, and 86.9%.
  • Entropy Sampling provides 70.3%, 83.88%, and 86.6%.
